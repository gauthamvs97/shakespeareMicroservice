Shakespeare Microservice

Developed by Gautham Varadarajan Shanmugaraj

Tech stack used: Docker, Python, Django Framework, Javascript, HTML and CSS

Instructions on File Format

  • The file uploaded must be a text file.
  • The text file must have all the topics / chapter names in all upper case at the beginning of the file.
  • Only the topics/chapters must be in all capitals
  • Upload the textfile and click on Submit Query
  • You will then be redirected to page with the Json Output

    You will be redirected to the output page
